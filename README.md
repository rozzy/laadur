# laadur

## Intro
When you are doing routine work, you may need a tool which will simplify your workflow. 
So became **laadur**.

Laadur creates a folder in your homepath (/Users/{user}/.laadur) and keeps there your templates.

## Using
Before using, install it:
```sh
gem install laadur`
```
